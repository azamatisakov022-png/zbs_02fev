# Главная публичная страница — дизайн-бриф для Claude Design

**Цель:** сделать публичную главную страницу АИС Минприроды КР **современнее и серьёзнее**. Сейчас используются плоские мультяшные иллюстрации и простые иконки — смотрятся дёшево для правительственного портала.

---

## Контекст

**Что это:** публичный портал ГП «Эко Оператор» / Минприроды Кыргызстана. Граждане заходят узнать о системе обращения с отходами, подать обращение, найти лицензию/переработчика на ГИС-карте.

**Кто смотрит:**
- Обычные граждане (информация, обратная связь)
- Бизнес (правила утильсбора, подача деклараций)
- Журналисты / эксперты / иностранные партнёры

**Язык:** русский основной + кыргызский + английский (через i18n).

## Визуальные требования

**Палитра (обязательная, брендовая):**
- Бирюзовый `#0e888d` — основной бренд
- Зелёный `#2D8B4E` — второй бренд
- Тёмно-серый `#415861` — основной текст
- Жёлтый `#fea629` — акцент (использовать осторожно)
- Нейтральные: slate-200 `#e2e8f0`, slate-50 `#f8fafc`, белый

**Тип/ощущение:**
- **Серьёзно, солидно, современно** — как сайты министерств экологии Европы, Сингапура, или серьёзных зелёных фондов
- НЕ коммерческое/рекламное/стартапское
- НЕ мультяшное/детское/игривое
- Чистые линии, много воздуха, качественная типографика

**Что использовать вместо мультяшек:**
- Абстрактные геометрические композиции (паттерны)
- Outline/duotone иконки в брендовых цветах (не цветные «стикеры»)
- Крупная типографика с номерами пунктов
- Градиенты от бренд-цвета
- Фото-фоны с приглушённым фильтром (природа / городская среда / переработка)
- Data-viz элементы (графики, инфографика) если уместно

**Шрифт:** системный sans-serif (уже используется Inter / SF / Segoe).

## Идеи для Секции 1 (направления деятельности)

Варианты которые можно попробовать:

**A) Minimalist list с номерами**
- Никаких иллюстраций
- Огромный номер направления (01, 02, 03...) и крупный заголовок
- Ниже тонкий текст + link «Подробнее»
- Разделение через тонкие линии
- Ощущение: журнал, не буклет

**B) Карточки с duotone outline-иконками**
- Вместо мультяшной иллюстрации — большая линейная иконка в брендовом цвете
- Иконки должны быть специфичны: монета для «Администрирование», здания+грузовик для «Инфраструктура», recycle-стрелки для «Переработка», лампочка/книга для «Просвещение», график для «Мониторинг»
- Плоские цветные плашки не использовать

**C) Карточки-статистики**
- Каждая карточка показывает **большое число** — то что достигнуто / то что делаем
- Пример: «Повышение уровня переработки» → «12% → 50% к 2030 г.»
- Ощущение: министерство отчитывается перед гражданами в KPI

**D) Photo-hero cards**
- Фото-фон с тёмным оверлеем
- Крупный заголовок белым + короткое описание
- Фото нейтральные (природа Кыргызстана, заводы переработки, мусоровозы)

## Идеи для Секции 2 (стратегия и миссия)

**A) Крупная типографика без иконок**
- 7 пунктов — просто крупные заголовки в 2-3 колонки
- Мелкий поясняющий текст
- Номера пунктов 01-07 большим серым
- Минимализм, без декора

**B) Timeline / дорожная карта**
- 7 пунктов в виде стратегии 2025→2030
- Горизонтальный timeline с маркерами
- Показывает движение вперёд

**C) Карточки с кастомными outline-иконками**
- Но каждая иконка РАЗНАЯ (не один и тот же квадратик с градацией)
- Brand color иконки, без заливки

**D) Две колонки: Миссия (слева крупно) + 7 пунктов списком справа**
- Текст миссии становится визуальным героем
- Справа — структурированный список с нумерацией

## Технические ограничения

- Vue 3 + TypeScript + Tailwind CSS
- Уже есть `container-main`, `section-title-underline` utility-классы

- Должно работать responsive: mobile / tablet / desktop
- Изображения через `<img src="/images/...">` (локальные файлы)
