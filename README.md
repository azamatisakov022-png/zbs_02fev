# АИС «Цифровой реестр отходов» - Frontend (Vue.js)

Фронтенд-приложение для автоматизированной информационной системы «Цифровой реестр отходов» МПРЭТН Кыргызской Республики.

## Технологии

- **Vue.js 3** - Прогрессивный JavaScript фреймворк
- **Vue Router 4** - Официальный роутер для Vue.js
- **TypeScript** - Типизированный JavaScript
- **Vite** - Быстрая сборка и dev-сервер
- **Tailwind CSS 3** - Утилитарный CSS-фреймворк

## Требования

- Node.js 18.x или выше
- npm 9.x или выше

## Установка

1. Клонируйте репозиторий и перейдите в папку проекта:

```bash
cd frontend-vue
```

2. Установите зависимости:

```bash
npm install
```

3. Скопируйте файл окружения:

```bash
cp .env.example .env
```

## Запуск

### Режим разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`.

### Предпросмотр сборки

```bash
npm run preview
```

## Страницы

| Путь | Название | Описание |
|------|----------|----------|
| `/` | Главная | Лендинг с информацией о системе |
| `/about` | О портале | Информация о целях и функциях системы |
| `/legislation` | Законодательство | Нормативные правовые акты |
| `/licenses` | Лицензии | Реестр лицензий на обращение с отходами |
| `/publications` | Публикации | Новости, отчёты и пресс-релизы |
| `/registries` | Реестры | Реестры плательщиков, переработчиков, полигонов |
| `/calculator` | Калькулятор | Расчёт утилизационного сбора |
| `/contests` | Конкурсы | Гранты и конкурсы в сфере экологии |
| `/reception-points` | Пункты приёма | Карта и список пунктов приёма отходов |
